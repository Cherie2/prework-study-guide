HTML

  • Hypertext Markup Language and is used to structure a web page and its content.
  • The head element contains information about the webpage.
  • The style element defines the CSS styles.
  • The base element defines the base URL.
  • The meta defines metadata like character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document/
  • The body element represents the visible content shown to the user.
  • Elements nestled inside other elements are called child elements.
  • The element that holds all the "children" is the parent element.
  • Element refers to the element from opening tag to closing tag.
  • Tag refers to only what appears inside the angle brackets.
  • The h1, h2, h3, h4, and h5 etc. elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or block text.
  • The ul ol and li elements represent unordered lists, ordered lists, and list items.
  • The img element contains info about images displayed.
  • The br element creates a line of empty space, or a line break between two blocks of content.
  • A link element creates links to webpages.

CSS

  • Cascading style sheets is a stylesheet language used to describe the presentation of a document written in HTML or XML
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • Selector is the HTML element name at the start of the ruleset. It defines the element(s) to be styled
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • In order to link changes in CSS to the HTML file you add to HTML file
  • A link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource.

Git

  • Git is a free and opensource version control systemt used for source code management.
  • The git status: checks what branch we are currently on.
  • The git checkout -b branch-name: creates a new branch and switches to it
  • The git add -A: adds changes to commit to all files.
  • The git commit -m: allows message with commit.
  • The git pull origin main: takes you to main branch/check for updates.
  • The git push origin feature/name of branch:allows to push changes from feature branch to main.

Javascript

  • Javascript is a powerful programming language that adds interactivity to your website.
  • Variables are containers that store values.
  • String is a variable that is a sequence of text; enclosed in single quote marks.
  • Number is a variable with no quotes around them.
  • Boolean is a variable that has a true/false value; doesn't need quote marks.
  • Array is a variable that has a structure that allows you to store multiple values in a single reference.
  • An array is a single variable that is used to hold a group of data; data related in some way.
  • Object is a variable that can be anything(everything in JavaScript is an object)
  • To declare a variable, place let before the name of the variable.
  • Comments are snippets of text that can be added along with code; use if no line breaks // This is a comment
  • Operator is a mathematical symbol that produces a result based on two values.
  • Conditionals are code structures used to test if an expression returns true or not; common form of conditionals is the if...else statement.
  • A semicolon at the end of a line indicates where a statement ends.
  • The script.js file is linked within the body element at the end of the HTML file.
  • JS code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • Functions do not automatically execute when the JavaScript file is run, rather you must call a function by name in order to execute it.
  • A function's name is followed by parentheses.
  • Adding */ before and */ after an if statement comments it out.
