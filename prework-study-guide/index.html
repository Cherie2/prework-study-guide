HTML

  • The head element contains information about the webpage.
  • style element defines the CSS styles.
  • base element defines the base URL.
  • meta defines metadata like character set, description, keywords, author, and viewport.
  • script element embeds client-side scripts in the HTML document/
  • The body element represents the visible content shown to the user.
  • Elements nestled inside other elements are called child elements.
  • The element that holds all the "children" is the parent element.
  • Element refers to the element from opening tag to closing tag.
  • Tag refers to only what appears inside the angle brackets.
  • h1, h2, h3, h4, and h5 etc. represent the level of heading a given text block represents.
  • p element represents a paragraph or block text.
  • ul ol and li represent unordered lists, ordered lists, and list items.
  • img contains info about images displayed.
  • br element creates a line of empty space, or a line break between two blocks of content.
  • a element creates links to webpages.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds changes to commit to all files.
  • git commit -m: allows message with commit.
  • git pull origin main: takes you to main branch/check for updates.
  • git push origin feature/name of branch:allows to push changes from feature branch to main.
